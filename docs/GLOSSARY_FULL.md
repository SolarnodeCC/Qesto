# Qesto — Glossary (Current)

_Hub: [Documentation map](./README.md)._

_Last verified: 2026-04-05 (UTC)_

## Core states
- **DRAFT**: session is editable.
- **LIVE**: session is active for participants.
- **CLOSED**: voting ended; results retained.
- **ARCHIVED**: historical, compliance-preserved state.

## Core actors
- **Presenter**: runs the live session.
- **Voter**: submits answers/reactions.
- **Viewer**: read-oriented team role.

## Platform terms
- **D1**: relational datastore.
- **KV**: low-latency key-value store.
- **Durable Object (SessionRoom)**: realtime coordination layer.
- **Vectorize**: semantic index for decision search.
- **Workers AI**: AI inference backend via `c.env.AI.run()`.

# Partner Tiers (v3.0)

| Tier | Requirements | Benefits |
|------|--------------|----------|
| **Registered** | Signed partner agreement | Directory listing, status API |
| **Built** | Certified integration (OAuth or Public API) | Co-marketing badge, sandbox |
| **Premier** | ≥3 production customers, SLA | Priority support, joint case studies |

## Status endpoints

- `GET /api/integrations/workday/status`
- `GET /api/integrations/jira/status`
- `GET /api/integrations/mattermost/status`

## Upgrade path

1. Register → receive sandbox API key (`Team` plan)
2. Ship integration → technical review
3. Premier → quarterly business review

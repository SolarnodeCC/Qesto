# v2.2.0 Release Candidate

## Highlights

- **LIVE engagement arc** (S26–S29): Quick Finger, Team Quiz, live leaderboard, badges, admin engagement analytics.
- **Enterprise foundation** (S31): AES-GCM integration token store, circuit breakers, enterprise permissions, GDPR trust page, compliance claim checker.
- **Release readiness** (S32): Rich session export (JSON/CSV/HTML), release health checklist, rollout/rollback plan, latency benchmark runbook.
- **CODE-SPLIT-01**: `functions/api/routes/sessions.ts` decomposed into `routes/sessions/{shared,public,crud,lifecycle,results,wizard,exports,index}.ts` with unchanged mount order and behavior.

## RC verification

Run the consolidated gate:

```bash
npm run check:rc
```

Accepted local exceptions (document in PR if they fail):

- `check:baseline` / `check:wrangler` when `CLOUDFLARE_API_TOKEN` is unset.
- Full-stack WebSocket smoke requires staging or local wrangler + D1 schema (see `knowledge-base/operations/GAM_STAGING_SMOKE_CHECKLIST.md`).

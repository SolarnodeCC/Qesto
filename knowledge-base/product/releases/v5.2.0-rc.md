**Status:** RC (staging)

## Highlights

- **DELIBERATE verifiable governance voting (foundation) — `DELIBERATE-RECEIPT-01`, P0, ADR-0049.**
  A new `session_mode = 'deliberate'` adds cryptographically-verifiable governance
  voting on the existing question engine:
  - **Coercion-resistant receipts.** Each ballot yields a receipt (ballot nonce,
    SHA-256 commitment, session fingerprint, the voter's own choice, verify path)
    — downloadable JSON/PDF, mobile-renderable. The receipt reveals only the
    voter's own choice; the published ledger reveals nothing about any other voter.
  - **Re-tallyable Merkle ledger.** Commitments form an order-independent Merkle
    tree; any observer can download the tally + commitment ledger and recompute
    the root, confirming `voteCount == commitmentCount`. A single tampered
    commitment changes the root and trips a forensics alert.
  - **Erasure-durable verification.** The ledger is anonymous by construction
    (`voter_hash` is a salted, per-session SHA-256 — no user id), so a voter can
    still verify their receipt **after deleting their account**, resolving the
    GDPR ⇄ immutable-ledger tension (ADR-0049).
  - Endpoints: `POST/GET /api/sessions/:id/deliberate/config`, `POST …/cast`,
    `POST …/verify`, `GET …/tally`. Team-tier gated (`verifiableVoting`).
- **SEC-VOTE-INTEGRITY-01 (P0).** Forgery/replay/coercion/de-anonymization review
  of the DELIBERATE surface (Pentest #5 pre-clearance) — see
  [`knowledge-base/security/SEC_VOTE_INTEGRITY_01_REVIEW.md`](../../security/SEC_VOTE_INTEGRITY_01_REVIEW.md).
- **Continuous collaboration → GA.** RETRO recurring workspaces (S85) + IDEATE
  board UX (`FE-IDEATE-BOARD-01`) complete the E85 continuous-collaboration epic.
- **IDEATE prioritization / facilitator board (`FE-IDEATE-BOARD-01`).** Dot-vote
  prioritization UI over the S85 IDEATE backend (SessionRoom WebSocket).
- **Agent facilitation GA hardening (`AGENT-FACILITATE-GA-01`).** GA-readiness
  pass on live AI facilitation — see
  [`AGENT_FACILITATE_GA_READINESS.md`](../../operations/monitoring/AGENT_FACILITATE_GA_READINESS.md).

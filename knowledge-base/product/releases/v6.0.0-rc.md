# v6.0.0-rc.1 — Release Candidate

**Status:** RC (staging)
**Sprint:** 89 (`claude/next-sprint-execution-c6enxt`)
**North star:** Gov full-ATO path · sovereign tenant tier · WCAG AAA GA · v6.0 release candidate
**GA target:** v6.0.0 at S90 (per [`SPRINT85_99_PLAN.md`](../planning/SPRINT85_99_PLAN.md) §Release map — certification bundle + DR drill + v5.x sunset)

This RC closes the post-v5.0 expansion arc (E81–E90). The six competitive epics
shipped across S84–S88 — TOWNHALL, STAGE, RETRO/IDEATE, DELIBERATE, EMBED, CANVAS +
CAPTIONS — are feature-complete; S89 hardens the security posture to RC quality and
lays the governance/compliance evidence for the S90 GA.

## Highlights

- **EMBED read-plane hardened — Pentest #5 gate closed (`PEN5-E1`/`E3`/`E4`).**
  The public widget data plane (`/api/embed/v1/*`, ADR-0050) now enforces a
  per-widget + per-origin rate limit (`429 + Retry-After`, tenant-isolated buckets,
  tighter budget on `/handshake`), resolves the handshake by canonical session id
  (no `id OR code` footgun), and returns an opaque `invalid_token` for all
  non-expiry token failures. This was the single named v6.0-rc availability blocker
  carried from S88; it is now **closed in shipped code** with a regression test
  (`tests/unit/embed-rate-limit.test.ts`). Overall Pentest #5 crit/high remains **0**.

- **CAPTIONS GA-eligible — `CAPTIONS-GA-01` WER sign-off.** The S88 live-captions
  pipeline (Whisper ASR + M2M100 MT on Workers AI, ADR-0051) is signed off for GA on
  the EN-source → {nl, es, de, fr} pair set, enforced by the eval gate against
  `CAPTION_WER_BAR = 0.25` (REV-10). No audio/transcript egress; transient-only.

- **WCAG AAA re-attested on captions + canvas.** AAA on the core flows (S88) is
  re-attested over the captions overlay and theme system shipped late in S88, holding
  the "core flows + captions/canvas AAA, broader app AA" claim for v6.0.

- **Gov Cloud full-ATO path + sovereign tenant tier — `ADR-0052`, `FEDRAMP-ATO-FULL-01`.**
  The authorization-boundary and control-inheritance design that moves Qesto from the
  S79 FedRAMP Moderate *control mapping* (ADR-0043) to a documented full-ATO **path**,
  plus the region-pinned sovereign data plane (irreversible D1 location hint, no
  cross-region replication, federation/egress exclusion). Documented as a path/target,
  **not** a granted authorization.

- **SOC 2 annual evidence refresh — `COMPLIANCE-SOC2-ANNUAL-01`.** v6.0-cycle control
  inventory snapshot and refreshed evidence (secret rotation, DR drill, circuit
  breakers, PII gate, Pentest #5), an internal evidence-tracking artifact.

- **DR drill — RTO ≤ 2h evidence.** Disaster-recovery drill establishing the v6.0
  recovery-time objective across D1/KV/DO/R2/Vectorize/secrets, with per-asset
  recovery procedures and an RPO analysis.

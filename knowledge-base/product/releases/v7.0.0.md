# v7.0.0 GA — Engagement Intelligence Network

**Sprint:** 99
**Date:** 2026-11-03
**Certification ADR:** [`ADR-0063`](../../adr/ADR-0063-v7-platform-certification.md)

## Highlights

The v7.0 arc (S91–S99) deepened and operationalized shipped surfaces into a
federated, AI-native engagement platform — the **Engagement Intelligence
Network**. GA at S99 certifies the platform and completes the XR beta launcher.

**Generally available (delivered across S91–S99):**
- **REACTIONS GA** — ephemeral high-throughput reaction layer (edge latency)
- **PULSE** — privacy-native HR/people-ops engagement analytics (k-anonymity, GDPR retention)
- **COPILOT GA** — supervised (L2) live AI facilitator co-pilot (Workers AI only)
- **LEARN** — LMS-embedded assessment via the EMBED SDK (LTI 1.1, grade passback)
- **SOVEREIGN+** — per-region edge residency + verifiable audit
- **CONNECT** — cross-tenant federated anonymous events (zero-knowledge across tenant boundaries)
- **STUDIO** — privacy-native AI session authoring (eval-gated, CANVAS theme-aware)

**Beta (feature-flagged, not part of the certified GA claim):**
- **XR** spatial session mode behind `beta-xr` (`BETA_XR_ENABLED`) — S99 completes
  the WebXR launcher (`FE-XR-LAUNCHER-01`) and the 2D graceful-degrade fallback
  (`XR-FALLBACK-01`). XR is opt-in and never gates voting.

## v6.x deprecation

Per ADR-0063, v7.0 GA is additive — no breaking change vs. v6.x. v6.x APIs enter a
24-month maintenance window (security/critical fixes only) ending **2029-11-03**,
with `Sunset-Date` headers on deprecated versions. v5/v4/v3 windows are unchanged.

## Upgrade

From v6.x: no action required — v7.0 is additive. To trial the XR beta, set
`BETA_XR_ENABLED=true`; the immersive launcher appears only when the session
advertises the `xr` capability and the device path is detected, and it never
blocks the standard 2D voting flow.

# ADR-0042: Native shell strategy (Capacitor + push only)

**Status:** Accepted (S73)  
**Date:** 2026-05-27

## Decision

Capacitor wraps the existing PWA for store-ready shells in S81+; S73–S80 ship push + offline only (no store submission).

## Consequences

- Developer portal v2 and push SLA are web-first.
- App Store assets deferred.

# Badges, Leaderboards, and Tournaments

Qesto can turn a session into a game — useful for trivia, training, kickoffs, and
high-energy events. Three features work together: **badges**, **leaderboards**, and
**tournaments**.

## Leaderboards

During scored activities (such as Team Quiz or a tournament), Qesto ranks participants in
real time. Points reward a mix of **speed** and **accuracy**, so quick correct answers rise to
the top. The leaderboard can be shown on the presenter screen so the room can follow along.

## Badges

When a session closes, Qesto awards badges based on how people took part. Badge types include:

- **First answer** — first to respond
- **Speedster** — consistently fast answers
- **Perfect trivia** — all quiz answers correct
- **Engagement** — answered every question
- **Leaderboard** — finished near the top
- **Streak** — a run of correct answers in a row
- **Consensus** — voted with the majority
- **Comeback** — climbed the ranking after a slow start

Badges are tied to a participant's activity in that session and are a light, fun way to
recognize involvement.

## Tournaments

A **tournament** runs a competitive activity as elimination rounds. After each round, the
lowest-scoring participants drop out and the field narrows until you have a winner —
bracket-style. This is great for large groups where you want a clear champion.

Related high-energy energizer formats include **Battle Royale** (everyone competes, the field
shrinks each round) and **Bracket** matchups. You set these up the same way as other
energizers while the session is in Draft.

Badges and session leaderboards work in scored activities. Tournament brackets are part of

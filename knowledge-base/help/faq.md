# Frequently Asked Questions

## General

**Q: What browser do I need?**
A: Qesto works on Chrome, Firefox, Safari, and Edge (desktop and mobile). We recommend the latest version for best compatibility.

**Q: Can I use Qesto on mobile?**
A: Yes! Qesto is fully responsive. Participants can join and vote from any device.

**Q: How many people can join a session?**
A: Free plan: 50 participants. Starter: 500. Team: Unlimited. You can always upgrade.

**Q: Is there a session time limit?**
A: No! Sessions can run as long as you need. However, we recommend closing after 2-3 hours for performance.

**Q: Can I edit questions after starting?**
A: No, questions are locked once the session goes live. You can add new questions, but can't edit existing ones.

**Q: Do I need a video call platform to use Qesto?**
A: No, Qesto is standalone. But it pairs great with Zoom, Teams, or Google Meet for facilitating discussions.

## Data & Privacy

**Q: Is my data encrypted?**
A: Yes, all data is encrypted in transit (HTTPS) and at rest.

**Q: Can I export my results?**
A: Yes! Download results as CSV from any closed session (Starter+ plans).

**Q: Do you store data outside the US?**
A: Qesto is hosted on Cloudflare's global edge network (data centers worldwide). All data complies with GDPR and CCPA.

**Q: How long do you keep my data?**
A: Data is retained for 1 year after a session closes. You can manually delete earlier.

## Usage

**Q: Can I use Qesto for free?**
A: Yes, the Free plan includes unlimited sessions with up to 50 participants and all 9 question types.

**Q: What's included in each plan?**
A: See our pricing page for a detailed comparison.

**Q: Can I cancel anytime?**
A: Yes, no lock-in contracts. You can downgrade or cancel anytime.

Have more questions? Contact support@qesto.cc

## Technical Troubleshooting

### Connection Issues

**Problem: "Connection lost" message**

1. Check your internet connection
2. Refresh the page (F5)
3. Try a different browser
4. Disable VPN/proxy if using one
5. Clear browser cache (Ctrl+Shift+Delete)

**Problem: Session link doesn't work**

1. Make sure the session is actually started (not in DRAFT)
2. Try the link in an incognito/private window
3. Check the URL is correct
4. Ask the host to restart the session

### Participant Issues

**Problem: "Participant limit reached"**

You've hit your plan's participant cap. Either:
- Upgrade to a higher plan
- Create a separate session for additional participants

**Problem: Responses not showing**

1. Refresh the page
2. Make sure you submitted your response (look for confirmation)
3. Check that the question is still open

## Performance

**Problem: Session is lagging**

1. Close other browser tabs
2. Check your internet speed (broadband recommended)
3. If 500+ participants: wait for responses to settle

**Problem: Audio/video not working**

Qesto doesn't include audio/video. For that, use Zoom, Teams, or Google Meet separately.

## Still Stuck?

Email support@qesto.cc with:
- Session ID
- What you were trying to do
- Error message (if any)
- Browser/device info

We'll respond within 2 hours.

## How to Use Energizers

### What Are Energizers?

Energizers are fun, low-pressure warm-up activities that run before your main questions. They help:

- Break the ice
- Test connection/device functionality
- Get everyone engaged
- Set a fun tone

### Available Energizers

#### Quick Finger

A speed game: See who can tap fastest.
- Participants tap as fast as possible for 10 seconds
- Leaderboard shows top tappers
- Great for remote teams to warm up

#### Team Quiz

A fun trivia game with instant scoring.
- Qesto gives you 3 fun team-focused questions
- Participants answer quickly
- Points awarded for speed + accuracy
- Leaderboard at the end

#### Word Cloud

Visual brainstorm: Type a word, watch a cloud form.
- Participants submit a single word
- Word size reflects frequency
- Fun way to see what's on everyone's mind
- Great for kickoff/retrospectives

### Adding Energizers

1. **In DRAFT mode**: Click "Add Energizer"
2. **Choose type**: Quick Finger, Team Quiz, or Word Cloud
3. **Configure** (if needed): Set time limits, questions
4. **Save**: Energizer is now in your session

### Running Energizers

1. **Start your session**: Energizers activate automatically
2. **Let it run**: Default is 1-2 minutes per energizer
3. **Watch live**: See responses flood in
4. **Auto-advance**: Or click "Next" manually

### Tips

- Use 1-2 energizers max (too many kills momentum)
- Great for sessions 30+ people
- Optional—skip them if your group knows each other
- Best for remote sessions (less need for in-person icebreakers)

Energizers make sessions more engaging and fun!

## Mobile and Accessibility Features

### Mobile Support

Qesto is fully responsive and works on all devices:

#### Phones
- Portrait and landscape orientation
- Touch-optimized buttons and inputs
- Fast loading on 4G/5G

#### Tablets
- Full-width layout for better readability
- Touch and stylus support
- Landscape recommended for presentations

#### Desktop
- Optimized for 1024px+ screens
- Works great on dual monitors

### Accessibility

Qesto meets WCAG 2.1 AA standards:

#### Keyboard Navigation
- Tab through all elements
- Enter to submit responses
- Escape to close modals
- Arrow keys for navigation

#### Screen Readers
- All buttons and links have labels
- Form fields are properly associated
- Live region announcements for updates

#### Color & Contrast
- High contrast colors (WCAG AA+)
- No color-only information
- Dark mode available

#### Text & Font
- Resizable text (Ctrl+ or Cmd+)
- Sans-serif font for readability
- 16px minimum font size

### Accessibility Settings

1. Click your profile → "Accessibility"
2. Options:
   - **Reduce motion**: Disable animations
   - **High contrast**: Increase color contrast
   - **Large text**: Increase default font size
   - **Dark mode**: Easy on the eyes

## Need Help?

Have accessibility concerns? Email support@qesto.cc—we're committed to inclusion!

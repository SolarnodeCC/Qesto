# Developer Community — Integrators (v3.0)

## Getting started

1. Create a Team plan workspace and API key at `/api/api-keys`.
2. Use **Public API v1** for session/results reads (`PUBLIC_API_V1.md`).
3. Use **Public API v2** for live WebSocket URLs (`PUBLIC_API_V2.md`, ADR-0024).
4. Official SDKs: `@qesto/sdk` (JS), `qesto-sdk` (Python).
